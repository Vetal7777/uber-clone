# Uber Clone

## Introduction

Welcome to the Uber Clone project! This is a ride-sharing application developed with Nuxt 3, Tailwind CSS, Prisma, Pinia, and Google Maps.

## Features

- User authentication
- Ride booking and tracking
- Real-time location updates
- Interactive maps using Google Maps API
- Responsive design for a seamless experience on various devices
